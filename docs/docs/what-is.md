# What is CC Workflow Studio

CC Workflow Studio is a VS Code extension that lets you design AI agent workflows on a visual canvas and export them as Markdown files that AI coding tools (Claude Code, GitHub Copilot, Cursor, Codex CLI, and others) understand natively.

> **You think visually. AI thinks in `.md`. CC Workflow Studio speaks both.**

## Why visual?

Prompt-only workflow authoring becomes hard to maintain as soon as you have more than a handful of steps, branches, or sub-agents. A canvas gives you:

- A clear overview of the entire flow
- Explicit branching, loops, and sub-agent boundaries
- Reusable nodes (Skills, MCP tools) you can drop in instead of re-prompting

## What you get

- **Visual editor** built on React Flow
- **12 node types** covering control flow, sub-agents, skills, MCP tools, and user prompts
- **8 export targets** for the major AI coding tools
- **Built-in MCP server** so AI agents can edit the canvas for you
