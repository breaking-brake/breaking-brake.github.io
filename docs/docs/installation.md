# Installation

CC Workflow Studio is distributed via the VS Code Marketplace and the OpenVSX Registry.

## VS Code Marketplace

Install [`breaking-brake.cc-wf-studio`](https://marketplace.visualstudio.com/items?itemName=breaking-brake.cc-wf-studio) from the Extensions view, or run:

```bash
code --install-extension breaking-brake.cc-wf-studio
```

## OpenVSX (VSCodium, Cursor, Windsurf, etc.)

Available as [`breaking-brake/cc-wf-studio`](https://open-vsx.org/extension/breaking-brake/cc-wf-studio) on OpenVSX.

## Requirements

- VS Code 1.80.0 or later (or any compatible editor)
- Node.js 18+ on your machine (only needed when running workflows that shell out to AI CLIs)
- Optional: Claude Code, GitHub Copilot Chat/CLI, Codex CLI, Cursor, Roo Code, Gemini CLI, or Antigravity — install whichever target agents you plan to export to

## After installation

Open the editor with **CC Workflow Studio: Open Editor** from the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`) or click the icon in the top-right corner of the editor.
