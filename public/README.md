# Public Assets

このディレクトリには、サイトで使用する静的アセットを配置します。

## アイコン画像の配置

以下のファイルを配置してください：

### ファビコン
- **ファイル名**: `favicon.ico` または `favicon.svg`
- **推奨形式**:
  - ICO形式の場合: 16x16, 32x32, 48x48のマルチサイズ
  - SVG形式の場合: スケーラブルなので1つのファイルでOK
- **用途**: ブラウザタブに表示されるアイコン

### ヘッダーロゴ
- **ファイル名**: `logo.png` または `logo.svg`
- **推奨サイズ**:
  - 高さ: 40-50px程度
  - 幅: アスペクト比に応じて（推奨: 150-200px程度）
- **用途**: サイトヘッダーに表示されるロゴ

## 配置後の確認

画像を配置したら、以下のURLでアクセスできます：
- ファビコン: `http://localhost:4321/favicon.ico`
- ロゴ: `http://localhost:4321/logo.png`
